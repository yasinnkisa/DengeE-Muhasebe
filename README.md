Bu proje, işletmelerin cari hesaplarını, stoklarını, banka/kasa hareketlerini, faturalarını ve giderlerini bulut tabanlı bir sistem üzerinden yönetmelerini sağlayan C# WinForms tabanlı bir masaüstü uygulamasıdır. Veri depolama için Google Firebase Realtime Database kullanılmıştır.

🚀 Öne Çıkan Özellikler
Bulut Veri Tabanı: Firebase entegrasyonu sayesinde veriler anlık olarak bulutta saklanır ve senkronize edilir.

Dinamik Stok Yönetimi: Ürünlerin kritik seviye kontrolü ve barkod tabanlı takibi.

Gelişmiş Fatura Takibi: Alış ve satış faturalarının dijital ortamda saklanması ve filtrelenmesi.

Finansal Analiz: Kasa ve banka hesapları arasında para transferi, giriş ve çıkış yönetimi.

Gider Raporlama: Harcamaların kategorize edilmesi ve yapay zeka destekli analiz raporlarının sunulması.

Güvenli Giriş Sistemi: Rol tabanlı yetkilendirme (Admin, Muhasebe, Gözlemci) ve E-posta ile şifre sıfırlama mekanizması.

🛠 Kullanılan Teknolojiler
Dil: C# (.NET Framework)

Veri Tabanı: Google Firebase Realtime Database (NoSQL)

Kütüphaneler: * FireSharp (Firebase REST API Wrapper)

Newtonsoft.Json (JSON Serialization/Deserialization)

Protokoller: SMTP (E-posta bildirimleri için)
